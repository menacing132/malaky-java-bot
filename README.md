# Malaky Discord Bot

Discord bot in **Java (JDA)** that shows Minecraft server status.

## Features
- Java IP
- Bedrock IP + Port
- Online / Offline status
- Players count
- Auto update every 5 minutes

## Setup
1. Create Discord Bot
2. Copy BOT TOKEN
